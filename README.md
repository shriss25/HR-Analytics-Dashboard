# 📊 HR Analytics Dashboard

**Power BI | DAX | Power Query | Excel**

An interactive HR Analytics Dashboard built using **Power BI** to analyze employee attrition, workforce demographics, salary distribution, and key HR metrics. The project demonstrates end-to-end data cleaning, KPI development, and dashboard design to generate actionable business insights from **1,470 employee records**.

---

## 📌 Project Overview

This project analyzes HR data to uncover patterns affecting employee retention and workforce performance. Using Power Query for data preparation and DAX for KPI calculations, the dashboard provides HR professionals with interactive insights into employee demographics, compensation, overtime, and attrition.

---

## 🔍 Key Insights

* Employees working **overtime** experience significantly higher attrition than employees without overtime.
* **Sales** has the highest departmental attrition rate across the organization.
* Employees earning **under $5K/month** show the highest attrition among all salary groups.
* Employees aged **18–25** have the highest attrition compared to older age groups.
* A custom **AttritionRiskFlag** identifies employees working overtime, earning under $5K, and reporting low job satisfaction. This high-risk segment experiences **48.1% attrition**, more than **three times** the overall company attrition rate.
